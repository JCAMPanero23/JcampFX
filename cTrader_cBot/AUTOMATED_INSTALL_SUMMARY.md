## How to Use (Quick)

### Option 1: Batch File (Easiest) ⭐

1. Open `D:\JcampFX\cTrader_cBot\` in Windows Explorer
2. Double-click `install_netmq.bat`
3. Wait for installation (10-30 seconds)
4. Script will show DLL path
5. In cTrader: **Manage References** → **Add Local File** → Navigate to DLL path
6. Done!

### Option 3: Python


## What the Scripts Do

1. **Download nuget.exe** (if not exists)
   - From: https://dist.nuget.org/win-x86-commandline/latest/nuget.exe
   - Saves to: `D:\JcampFX\cTrader_cBot\nuget.exe`

2. **Create packages directory**
   - Creates: `D:\JcampFX\cTrader_cBot\packages\`

3. **Install NetMQ 4.0.1.13**
   - Runs: `nuget.exe install NetMQ -Version 4.0.1.13 -OutputDirectory packages`
   - Downloads: ~2 MB
   - Installs to: `packages\NetMQ.4.0.1.13\`

4. **Show DLL path**
   - Displays: `packages\NetMQ.4.0.1.13\lib\net47\NetMQ.dll`
   - This is the file you add to cTrader references

## Troubleshooting

### Batch file doesn't run
**Solution:** Right-click → **Run as Administrator**

### "Execution Policy" error (PowerShell)
**Solution:**
```powershell
powershell -ExecutionPolicy Bypass -File install_netmq.ps1
```

### Python script fails
**Solution:** Ensure Python 3.6+ installed:
```bash
python --version  # Should show 3.6 or higher
```

### nuget.exe download fails
**Solution:**
1. Download manually: https://dist.nuget.org/win-x86-commandline/latest/nuget.exe
2. Save to `D:\JcampFX\cTrader_cBot\nuget.exe`
3. Re-run script

### NetMQ install fails
**Solution:** Check internet connection, or if behind corporate proxy:
```bash
nuget.exe config -set http_proxy=http://proxy.company.com:8080
```
